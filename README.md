## Usage

You need to first open [HERE](https://beta.openai.com/account/api-keys) to get a valid API Key.

```sh
# Install
pip install git+https://github.com/jaypei/chatgpt-cli

# Start a chat session
chatgpt-cli chat

# or make an alias
alias chat="chatgpt-cli chat"
chat

# have fun
```
